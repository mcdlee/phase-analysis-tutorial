講到 Gated imaging 之前

我們先講波

波的性質 (小學自然)

  • 振幅
  • 頻率
  • 波形

以聲音 (一種機械波) 為例

  • 振幅: 音量
  • 頻率: 高低音
  • 波形: 音質

波可以視為一種週期性函數

  • 以空氣中的聲波為例
    • 空間位置對空氣疏密度的函數
    • 時間對空氣疏密度的函數
  • 以繩波為例
    • 位置對繩段偏移程度的函數
    • 時間對繩段偏移程度的函數

所有的週期性函數都可以透過

傅立葉轉換

變成很多個 sine wave 總和 - 稱為頻譜

  • 不同的樂器有不同的波型，是因為有不同比例的泛音，所以我們可以聽到不同的音色。
    • 各種樂器的波型請看台灣師大物理系黃福坤老師的網站
  • 不同音素也有不同的波型，所以我們可以聽得懂彼此的話語。
    • 這也是為什麼重聽的喪失的聽力是以高頻的為先，卻會造成生活上的困擾。
  • 其實頻率分析離我們並不遠。
    • 影音播放軟體的長條圖
    • 內耳的耳蝸

Phase Image (以 MUGA 為例)

  • First-order Fourier fitting is performed for each pixel to create the amplitude images.
    • The pixel values less than 10% of the maximun are assumed to be background and replaced with 0.
  • Assuming that the pixel values in phase difference 0 status is 180, a phase image is created in the range from 0 to 360.
    • The pixel assumed to be background in the amplitude image are set to 0.

Source: Siemens workstation 的說明

以 Gated SPECT 為例
